Warn me is a plugin, which lets you to warn people and uses MySQL for storage.

## Basic commands

Warn a player

```markdown
/warn add <nickname> <reason>
```

Show player's informatrion

```markdown
/warn info <nickname>
```

Clear player's warns

```markdown
/warn clear <nickname>
```
